# Autonomous Digital Organization Platform

A self-sustaining digital entity built on a self-hosted, MCP-native governed operations layer.

## Overview

The platform provides **token independence**, **recursive self-improvement**, **predictive governance**,
**agent-owned assets**, **self-negotiating treaties**, **digital twin simulation**, and **autonomous incident resolution** —
all through a governed MCP tool layer with uniform policy enforcement and signed execution receipts.

## Key Capabilities

- **98 MCP tools** across all operational domains
- **5 agent runtimes**: Claude Code, OpenCode, Direct MCP, Workflow Engine, Collaborator Desktop IDE
- **6 operational pillars**: Guardrails, Sandbox, HITL, Registry, War Room, Monitoring
- **Governance OS**: Constitutional policy enforcement with auto-amending case law
- **Treaty-Based Federation**: Cross-instance governance with dual-signed ledgers

## Quick Start

```bash
# Install
curl -fsSL https://raw.githubusercontent.com/geludobre/sovereign-agentops/main/scripts/install.sh | bash

# Start the platform
./scripts/start.sh

# Check health
curl https://localhost/api/health
```
