# Contributing to Sovereign AgentOps

## Reporting Issues

- Check existing issues before filing a duplicate
- Include platform version, runtime (Claude Code, OpenCode, Direct MCP, Workflow Engine), and steps to reproduce
- For suspected security vulnerabilities, do not file a public issue. Email the maintainers directly at the address listed in the README.

## Submitting Changes

1. Fork the repository and create a feature branch from `main`
2. Make your changes with clear commit messages
3. Run the test harness and ensure all checks pass
4. Submit a pull request with a description of the change and its motivation
5. Maintain sign-off on all commits (Developer Certificate of Origin)

## Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/) Code of Conduct. By participating, you agree to uphold its standards. Report unacceptable behavior to the maintainers.
